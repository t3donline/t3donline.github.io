Coca Cola Sales Dashboard

Dashboard report for Coca Cola Sales Report Analysis
Report Showing some key insight for the Coca Cola Sales and also using Q&A Query Style

Objective:

- to provide interactive Powerbi dashboard where user can ask question directly using QA style , such as: What is the best selling state? , best selling brand, etc.
- to show some insight on operating profit and how to increase the operating profit ( We will use the Key influencers for this case)
- to Provide Sales By State information ( We Will use map chart for this case)
- to show more financial information for the sales
- Report can be filtered by certain date range

Data source: Data in Excel file format


Using Q&A Query Style
