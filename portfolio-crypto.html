Cryptocurrency Portfolio

Data analytics report for Cryptocurrency investment.
Main Objective: to make it easier and faster to get a summarize information of crypto portfolio investment.
Latest crypto Price is imported directly from coinmarketcap.com using API ( Application Programming Interface) key to show the realtime latest market price.
Report showing some key financial information :
- Profit, Holdings, Worst and Top Performer
- Coins Allocation in total portfolios, acquisition cost with profit/loss , ROI
- Detail list of cryptocurrency with number of coins bought information, dollars spent, current price and current value, ROI
